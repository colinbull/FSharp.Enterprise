F# Enterprise Web - Channels

FSharp.Enterprise.Web.dll provides an implementation of a channel to provide communication over http. the HttpChannel<'a> wraps the System.Net.Http.WebClient

1: open FSharp.Enterprise
2: open FSharp.Enterprise.Channel
3: open System.Net.Http.Formatting

To use the Http channel we need to provide a (MediaTypeFormatter)[http://msdn.microsoft.com/en-us/library/system.net.http.formatting.mediatypeformatter(v=vs.108).aspx]

1: let formatter = new JsonMediaTypeFormatter()
2: formatter.SerializerSettings <- Serialisation.Json.settings
Here we use the default Json formatter but we override the serialiser settings with ones provided by the core library (see. here)[Core.Json.html]

1: let http = new Http.HttpChannel<_>("Example", new System.Net.Http.Formatting.JsonMediaTypeFormatter())
