F# Enterprise Library

F# Enterprise: Library for Enterprise development

The F# Enterprise library collection aims to simplify common tasks encountered when developing enterprise applications, by providing a full set of helpers for:

  • Asyncronous / Agent Programming
  • Scheduling
  • Serialisaton
  • Messaging
  • Web
  • and much more...

This library is intended to complement other F# libraries, such as FSharpx and FSharp.Data

F# Enterprise consists of the following set of dlls.

FSharp.Enterprise.dll

This provides a core set of modules,

  • Agent
  • Async
  • Caching
  • Channel
  • DateTime
  • Environment
  • FileSystem
  • Html
  • IO
  • Json
  • Net
  • Process
  • Regex
  • Scheduling
  • Security
  • Serialisation
  • String
  • Uri
  • Xml

FSharp.Enterprise.RabbitMq.dll

This provides a set of types that implement common messaging patterns over RabbitMQ.

  • RPC
  • Pubish - Subscribe
  • Channel

FSharp.Enterprise.Web.dll

This provides a set of modules and extensions that helps with web programming.

  • Channels
  • WebAPI
